## 📝 购买步骤

### 步骤1：登录腾讯云
访问：https://console.cloud.tencent.com/

### 步骤2：进入轻量数据库页面

**方法A**（推荐）：
1. 控制台首页 → 顶部搜索"轻量数据库"
2. 点击进入轻量数据库控制台
3. 点击"新建"或"创建实例"

**方法B**：
1. 左侧导航 → "轻量应用服务器"
2. 选择"数据库"标签
3. 创建轻量数据库

### 步骤3：选择配置

**关键配置项**：
- ✅ **数据库类型**：PostgreSQL
- ✅ **地域**：选择与服务器相同地域（推荐，降低延迟）
- ✅ **规格**：1核1G内存（最低配置）
- ✅ **存储**：20GB SSD
- ✅ **计费方式**：
  - 月付：¥35/月
  - 年付：¥168/年（**推荐，约¥14/月**）
  - 三年：¥378（最优惠）

### 步骤4：设置密码和网络

- **数据库密码**：设置root用户密码（请务必记住！）
- **网络类型**：
  - **内网**（推荐）：延迟<5ms，免费
  - **外网**：需要开启外网访问，有流量费用

### 步骤5：确认购买

- 检查配置和价格
- 确认购买

## 📋 购买后操作

### 1. 获取连接信息

在控制台的"实例详情"中获取：

```
数据库地址：xxxxx.tencentcdb.com  （外网）
内网地址：10.0.x.x                （内网，推荐）
端口：5432
用户名：root（默认）
密码：你设置的密码
```

### 2. 配置后端环境变量

编辑 `backend/.env`：

```env
# 使用内网地址（推荐，延迟最低）
DB_HOST=10.0.x.x              # 内网IP（从控制台查看）
DB_PORT=5432
DB_USER=root
DB_PASSWORD=你的密码
DB_NAME=naibao
DB_SSLMODE=disable             # 内网可以不用SSL

# 或使用外网地址
# DB_HOST=xxxxx.tencentcdb.com  # 外网地址
# DB_SSLMODE=require            # 外网需要SSL
```

### 3. 初始化数据库

通过腾讯云控制台的"数据库管理"工具执行SQL：
- 执行 `database/schema.sql`
- 执行 `database/init_data.sql`

## 🆘 如果找不到轻量数据库选项

### 可能的情况：

1. **地区限制**：某些地区可能没有轻量数据库，尝试选择：
   - 北京
   - 上海  
   - 广州

2. **产品名称变化**：可能显示为：
   - "轻量数据库"
   - "轻量应用数据库"
   - "Lighthouse Database"

3. **替代方案**：如果确实找不到轻量数据库，可以使用：
   - **云数据库PostgreSQL基础版**（¥300-400/月，稍贵但功能更多）
   - 直接在轻量应用服务器上安装PostgreSQL（需要自行维护）
