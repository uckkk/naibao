# 🚀 快速开始测试（3步）

## 步骤1: 在本地Mac上启动前端

打开Mac终端，执行：

```bash
cd "<你的项目目录>/naibao/frontend"
npm install
npm run dev:h5
```

## 步骤2: 查看Network地址

终端会显示类似：
```
  Network: http://192.168.1.100:8080
```

## 步骤3: 手机浏览器访问

1. 确保手机和Mac在同一WiFi
2. 在手机浏览器输入：`http://192.168.1.100:8080`（替换为你的Network地址）
3. 开始测试！

---

## ⚠️ 重要提示

- ✅ **前端在本地Mac运行**（不是服务器）
- ✅ **后端在服务器运行**（已配置好，无需操作）
- ✅ **手机访问本地Mac的前端服务**

---

## 🐛 如果遇到问题

### 问题1: npm命令不存在

**解决**：安装Node.js
```bash
# 使用Homebrew安装
brew install node
```

### 问题2: 手机无法访问

**检查**：
- 手机和Mac是否在同一WiFi
- Mac防火墙是否允许8080端口
- Network地址是否正确

### 问题3: API请求失败

**检查**：
- 服务器是否运行（已运行✅）
- API地址配置是否正确（已配置✅）
