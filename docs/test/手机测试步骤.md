# 📱 手机测试步骤（最简单方式）

本仓库已将“手机测试”相关说明收敛为一份可长期维护的文档，请以此为准：

- `docs/test/手机测试指南.md`

本文件保留用于兼容旧链接，不再单独维护细节。

## 快速开始（3步）

1) 配置前端 API 地址（本机私有，不提交）

```bash
cp "frontend/.env.example" "frontend/.env.local"
$EDITOR "frontend/.env.local"
```

2) 启动前端（H5）

```bash
cd "frontend"
npm ci
npm run dev:h5
```

3) 用手机浏览器打开终端输出的 `Network` 地址进行测试。

然后在微信开发者工具中打开 `frontend/dist/dev/mp-weixin`

---

**现在就可以开始测试了！** 🎉
