## 🐛 如果遇到问题

### 问题1: 端口被占用

修改vite.config.js中的端口：
```javascript
server: {
  port: 3000, // 改为其他端口
}
```

### 问题2: 手机无法访问

**检查**：
- 手机和Mac是否在同一WiFi
- Mac防火墙是否允许该端口
- Network地址是否正确

### 问题3: API请求失败

**检查**：
- 服务器是否运行（已运行✅）
- API地址配置是否正确（检查 `frontend/.env.local` 的 `VITE_API_BASE_URL`）
